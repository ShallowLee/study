#HTTP
>http的本质就是从互联网的某处获取资源（文档，图片，视频）

#HTTP的实现方式
>我们通过浏览器浏览url上的资源

##url
>全球唯一的，定位资源用的一个地址。

###组成方式
![http1](http1.png)

1. 协议名
2. 服务器地址（可以是域名也可以是ip地址）
3. 端口号（默认为80）
4. 目录
5. 查询字符

##浏览器是如何通过url来访问资源的
1. 查看协议名（这里探讨只http）
2. 寻找服务器地址（解析域名）
3. 寻找端口号
4. 查看目录
5. 解析服务器端返回回来的数据（html）并渲染到屏幕上。
